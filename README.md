# 🚀 Deepak Link Hub

A simple and stylish **Linktree-style personal portfolio page** to showcase all my important profiles and links in one place.

---

## 🌐 Live Demo

👉 https://d33pAk1106.github.io/my-links/

---

## 📌 Features

* 🔗 Centralized links (CodeForces, GitHub, LeetCode, LinkedIn, Facebook)
* 🎨 Modern UI with smooth animations
* 🌙 Dark Mode toggle
* 🖼️ Animated background effects
* 💡 Responsive and minimal design

---

## 🛠️ Built With

* HTML5
* CSS3 (Animations & Variables)
* JavaScript (Dark Mode Toggle)

## ⚙️ How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/d33pAk1106/my-links.git
   ```

2. Open the folder:

   ```bash
   cd my-links
   ```

3. Open in browser:

   * Double click `index.html`
   * OR use Live Server in VS Code

---

## ✏️ Customization

* Update your links inside `<a>` tags
* Change name/title in `<h1>`
* Modify colors using CSS variables
* Replace background image URL if needed
